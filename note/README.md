# 📝 SmartNote – Full Stack Notes App

A modern, responsive **MERN Stack Notes Application** that allows users to create, manage, and organize their notes efficiently with authentication and real-time updates.

---

## 🚀 Live Demo

🔗 Frontend: https://smartnote-self.vercel.app/
🔗 Backend: https://smartnote-31wh.onrender.com/

---

## ✨ Features

* 🔐 User Authentication (Login / Signup)
* 📝 Create, Edit, Delete Notes
* 📌 Pin Important Notes
* 🔍 Search Notes
* 🏷️ Tag System
* 📱 Fully Responsive UI (Mobile + Desktop)
* ⚡ Fast & Clean User Experience
* 🔔 Toast Notifications

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

### Deployment

* Frontend: Vercel
* Backend: Render

---

## 💡 Key Learnings

* Building full-stack applications using MERN
* Implementing authentication with JWT
* API integration with Axios
* Responsive UI design using Tailwind CSS
* Deployment using Vercel & Render
